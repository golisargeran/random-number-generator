# Random Number Generator

A simple and responsive random number generator built with HTML, CSS, JavaScript, and Bootstrap.

## Features
- Generate 3 random numbers with one click
- Clean and responsive UI using Bootstrap
- Simple DOM manipulation
- Beginner-friendly structure

## Technologies
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (DOM events)

## How to Run
1. Clone the repository:
   git clone https://github.com/golisargeran/random-number-generator.git
3. Open `index.html` in your browser.
